# Frappe Painter

Comprehensive UI/UX Theme Customizer for Frappe/ERPNext v15.

## Installation
```bash
bench get-app https://github.com/krunal1904/frappe_painter.git
bench --site yoursite.com install-app frappe_painter
```

## Features

- Enable/Disable theme toggle
- Navbar, Sidebar, Buttons, Body, Tables, Forms, Widgets customization
- Login page customization
- 7 built-in templates (Business, Hospital, Audit, ERPNext, Government, Retail, Tech Startup)
- Custom CSS support
